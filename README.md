# Premium Developer Portfolio - Prem Bhoi

A modern, professional, and visually stunning personal portfolio website designed for **Prem Bhoi**, an aspiring Software Developer and Computer Science Engineering student.

This portfolio is built using a dark glassmorphism theme, featuring neon indigo, purple, and blue gradient accents, smooth responsive layouts, and interactive developer elements.

## 🚀 Features

- **Futuristic Dark Theme**: Sleek background glow blobs and custom gradient styling.
- **Glassmorphism Design System**: High-quality frosted-glass cards (`backdrop-filter`) and border highlights.
- **3D Device Mockups**: Displays actual project screenshots inside custom 3D web-browser and mobile-phone frames.
- **Typewriter Effect**: Dynamic header subtitles that cycle through engineering roles.
- **Interactive Statistics**: Custom SVG radial gauge animating the CGPA score dynamically.
- **Scrollspy Navigation**: Sticky navbar with active section indicators and scroll progress bar.
- **Animations**: Entrance scroll reveal fades and neon glowing cards on hover.
- **Viewable Credentials**: Instant access buttons to view/download original internship certificate PDFs and resume.

## 🛠️ Tech Stack

- **Structure**: HTML5 (Semantic tags, SEO configurations)
- **Styling**: Vanilla CSS3 (Custom variables, flexbox, grid, keyframe animations)
- **Interactivity**: Vanilla JavaScript (ES6+, Intersection Observer API)
- **Icons**: FontAwesome v6

## 💻 Local Setup & Running

To run this project locally, simply clone the repository and open the HTML file:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Prembhoi377/portfolio.git
   ```
2. **Open index.html**:
   - Double-click `index.html` to open it in your web browser.
   - Or, run a local development server using:
     ```bash
     npx http-server
     ```
     or
     ```bash
     python -m http.server 8000
     ```
